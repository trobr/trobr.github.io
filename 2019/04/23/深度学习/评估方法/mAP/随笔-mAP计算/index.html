2. 目标检测中评估方法

2.1 关于正负样本

上述模型评估方法可以较好的解决分类问题中的模型分类问题，但是目标检测中，网络的输出不仅包含目标的分类，还包含目标的位置信息。如果需要在目标检测中应用查准率和召回率评估模型，需要对查准率和召回率进行些许改变。
分类问题中，一个对象属于某一类别，则对于这个类别（class，如猫、狗、杯子等）来说，该对象就是该类别的正样本（positive），否则则为负样本（negative）。
而在目标检测问题中，正负样本量化为：对于某一个候选框，计算其与某一类别（class）ground truth的IOU（交并比），同时设定一个阈值如0.5，如果候选框与ground truth的IOU大于阈值，则认为该候选框为该类别的正样本，否则为负样本；如果类别的某一个ground truth有多个候选框与其对应，则选择这些候选框与ground truth的IOU最大的候选框作为正样本，其余设置为负样本。
