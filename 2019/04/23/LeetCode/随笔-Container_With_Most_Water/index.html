0x00 题目描述

Given n non-negative integers a1, a2, …, an , where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.

给定 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

0x01 解题思路

题目实际上求解一个矩形面积，矩形面积由宽高决定，因此要使得面积最大，应该是宽和高竟可能大。

  1. 对于宽，即坐标的差值，当两点位于首尾时最大
  2. 对于高，决定高的实际上是两条边中较小的那一条

因此初始化时将两个初始点分别定位到首尾位置，使得矩形宽竟可能大，其次对于高，因为围成矩形的高由较短的边决定，因此我们从首尾位置开始，不断使两个初始点靠拢，这个过程中，保留值大的那一边。

0x02 例子

对于下面的数组height：

1 8 6 2 5 4 8 3 7

长度为9，因此初始化左右两个点left,right分别为

[0, 8]

则面积为：

max(height[0], height[8]) * (8 - 1) >> 48 >> s0

这时为了继续遍历，应减小left,right的差值，因为在这次的面积计算中，限制了面积的是两条边中较小的边height[0] >> 1，如果继续保留较小的这条边，那么后面另外一条边值是多少，最终组成的矩形高都小于等于height[0]，因此面积永远小于等于s0，所以我们需要保留大的一条边，于是有：

left += 1;
right = right;

0x03 代码

class Solution {
    public:
        int maxArea(vector<int> &height) 
        {
            //
            int max_area = 0;
            for (int s = 0, len = height.size() - 1, e = height.size() - 1; s < e;) {
                int min_val = 0;
                if (height[s] > height[e]) {
                    min_val = height[e];
                    e -= 1;
                }
                else {
                    min_val = height[s];
                    s += 1;
                }

                max_area = max((e - s + 1) * min_val, max_area);
                cout << "s:" << s 
                    << "\te:" << e
                    << "\tmax:" << max_area << endl;
            }

            return max_area;
        }
    };
