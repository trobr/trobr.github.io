vscode中导入自定义python包时显示unslove_import问题

Posted on 2019-04-23 | In VSCode | Visitors:

简介——————————————-

  • 问题描述
  • 解决方法
  • 参考

问题描述

使用vscode编写python代码时， 如果在子文件夹下定义了python包，然后在其他文件中使用import导入时，vscode会在包名下面生成绿色波浪线，提示unslove import问题
如下所示：

解决方法

详见参考

  1. 在工作目录下.vscode文件夹中创建.env文件
  2. 在.env文件下写入以下PYTHONPATH=codeFolder,其中codeFolder是工作区所在的文件夹名字
  3. 在setting.json中添加"python.envFile": "$${workspaceFolder}/.vscode/.env"
