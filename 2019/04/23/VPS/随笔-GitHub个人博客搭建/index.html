Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上。
借助于Hexo我们可以快速的在GitHub上搭建我们自己的个人博客。
本篇文章介绍了如何安装使用Hexo和GitHub搭建自己的个人博客。此外还有介绍了一些优化的小工具。
