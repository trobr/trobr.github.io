Python-OpenCV

Posted on 2019-04-23 | In Python | Visitors:

OpenCV是一个基于BSD许可（开源）发行的跨平台计算机视觉库，可以运行在Linux、Windows、Android和Mac OS操作系统上。它轻量级而且高效——由一系列 C 函数和少量 C++ 类构成，同时提供了Python、Ruby、MATLAB等语言的接口，实现了图像处理和计算机视觉方面的很多通用算法。

1 常用操作

1.1 导入包

import cv2

1.2 OpenCV读取图片

src = cv2.imread(r’1.jpg’)

1.3 OpenCV访问一个像素

src[height, width]

如果是一个RGB图片，则执行上述命令会输出一个一行三列的numpy数组，RGB通道值以BGR的顺序存储

1.4 输出图片维度

print(src.shape)

numpy中维度信息组合方式为(height, width, channel)，如果图片宽高为 800*300 的RGB图片，输出维度为(300, 800, 3)

1.n OpenCV设置图片ROI

roi = src[height_min:height_max, width_min:width_max]

2 旋转变换

2.1 透视变换

pts1 = np.float32([179, 225], [136, 501], [765, 343], [762, 554])
pts2 = np.float32([[0, 0], [0, 300], [800, 0], [800, 300]])
print(pts1)
print(pts2)
M = cv2.getPerspectiveTransform(pts1, pts2)
dst = cv2.warpPerspective(src, M, (800, 300))

这里的坐标要以[x, y]的形式存储，而不是[height, width]
对于四个坐标的顺序没有要求，只要两组坐标pts1和pts2的顺序相同即可，即如果pts1是以左上->左下->右上->右下的顺序，那么pts2也要以这个顺序。
