装饰器与装饰器符号@

1. 装饰器的作用

装饰器的作用就是在已知的函数上通过另一个函数为这个已知函数添加一些额外的功能，大多数情况下添加的这些额外功能是一些通用的功能，这样不仅可以在不修改现有函数的情况下添加一些新功能，而且这些额外功能就不需要在每个需要修饰的函数中重复添加，可以节省代码空间。

2. 例子

假设有现在有两个函数add和sub分别求两个数的和与差，在以前的生产环境中，不会出现结果为负的情况，代码没有进行边界检查但是新环境下，必须加上边界检查以防止出现问题，在不改变add和sub内部代码的情况下添加边界检查就需要使用装饰器。
两个原始函数定义如下：

def add(a, b):
    return a + b
def sub(a, b):
    return a - b
  1. 先考虑最简单的情况

再定义一个边界检查函数，函数接受一个值，返回边界处理后的结果

def abs(x):
    return x if x >= 0 else 0
abs(add(1, -2))
abs(sub(3, 5))
  1. 使用装饰器

虽然上述代码能够解决边界检查的问题，但是需要额外调用一个函数，实际使用时可能会忘记，如何才能在不修改add的情况下直是这些函数具有边界检查的效果呢。———装饰器

def abs_decor(fun):
    def inner(a, b):
        result = fun(a, b)
        return result if result >= 0 else 0
    return inner

add = abs_decor(add)
sub = abs_decor(sub)

abs(add(1, -2))
abs(sub(3, 5))

使用装饰器实际上是就是用装饰器装饰后的函数代替原函数
在Python中函数也是作为一个对象来处理，因此可以将函数作为另一个函数的参数 ——— 详情见参考
装饰器接受需要装饰的函数作为参数，在装饰器内定义一个新函数，对需要装饰的函数的结果添加额外的操作，再返回添加操作后的结果。（注意这个结果是装饰器内新定义的函数返回的，而不是装饰器函数返回），装饰器函数将内部新定义的函数作为返回值返回，这样在外部只需要

add = abs_decor(add)
sub = abs_decor(sub)

这样操作即可得到经过装饰器装饰后的原函数。

  1. 装饰器符号@

装饰器符号@实际上就是

将操作：

def add(a, b):
    return a + b

add = abs_decor(add)

简化为：

@abs_decor
def add(a, b):
    return a + b
